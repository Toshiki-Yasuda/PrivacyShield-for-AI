# PrivacyShield for AI

企業向けAI利用時の個人情報保護ツール。AIサービス（Claude.ai）利用時に個人情報を自動検知・マスキングし、安全なプロンプト入力を支援するChrome拡張機能です。

## 機能

- **個人情報の自動検知**: 氏名、メールアドレス、電話番号、住所、企業名を自動検出
- **リアルタイムマスキング**: 入力と同時にマスキング処理
- **サイドパネルUI**: 左右分割表示でオリジナルとマスキング後のテキストを比較
- **復元機能**: マスキングしたテキストを元に戻す
- **カスタムパターン**: 独自の正規表現パターンを追加可能
- **完全ローカル処理**: すべての処理はブラウザ内で完結、外部への情報送信なし

## インストール方法

1. このリポジトリをダウンロードまたはクローン
2. Chromeで `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」を有効化
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. `PrivacyShield-for-AI` フォルダを選択

## 使用方法

### 基本操作

1. Claude.aiにアクセス
2. 拡張機能アイコンをクリックして「サイドパネルを開く」
3. 左側のテキストエリアにマスキングしたいテキストを入力
4. 右側に自動でマスキング結果が表示
5. 「コピー」ボタンでマスキング済みテキストをコピー

### テスト用テキスト

```
山田太郎様
連絡先: 090-1234-5678
メール: yamada@example.com
住所: 東京都渋谷区〇〇町1-2-3
株式会社テスト
```

### カスタムパターンの追加

1. 拡張機能の設定画面を開く
2. 「カスタムパターン」セクションで「追加」をクリック
3. パターン名、ラベル、正規表現を入力
4. テスト機能で動作確認
5. 「保存」で設定を保存

## 対応サービス

- **Phase 2 (予定)**: ChatGPT

## デフォルトマスキングパターン

| パターン | 説明 | ラベル |
|---------|------|--------|
| 氏名 | 日本語の姓名（山田太郎様など） | [Person_A] |
| メール | メールアドレス | [Email_A] |
| 電話番号 | 日本の電話番号（090-1234-5678） | [Phone_A] |
| 住所 | 都道府県から始まる住所 | [Location_A] |
| 企業名 | 株式会社・有限会社など | [Company_A] |

## 技術仕様

- **Manifest Version**: 3
- **対応ブラウザ**: Chrome（Manifest V3対応）
- **データ保存**: chrome.storage.sync（設定の同期）

## プライバシー

- すべての処理はローカルで実行
- 外部サーバーへのデータ送信は一切なし
- マスキングマッピングはセッション内のみ保持
